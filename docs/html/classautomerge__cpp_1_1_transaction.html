class Document

Detailed Description

A mutation interface for modifying a Document.

Transactions are created exclusively by Document::transact() or Document::transact_with_patches(). All mutations to a document must go through a transaction, which ensures atomicity: either all operations in the transaction are applied, or none are.

doc.transact([](auto& tx) {
tx.put(root, "name", std::string{"Alice"});
tx.put(root, "age", std::int64_t{30});
});
