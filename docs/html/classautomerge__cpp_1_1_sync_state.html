Detailed Description

Per-peer synchronization state machine.

Create one SyncState per peer you are synchronizing with. Pass it to Document::generate_sync_message() and Document::receive_sync_message() to drive the sync protocol.

SyncState can be persisted with encode()/decode() for resumable sync.
