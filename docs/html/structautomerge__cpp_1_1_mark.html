Detailed Description

A rich-text mark spanning a range of a text or list object.

Marks annotate ranges with named attributes (e.g. "bold", "italic", "link"). They survive insertions, deletions, and merges because they are anchored to element OpIds internally, not indices.

Apply marks with Transaction::mark() and query them with Document::marks() or Document::marks_at().
